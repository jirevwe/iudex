# Changelog

## 1.0.0 - 2026-02-09

### Added

- Core API testing framework with TypeScript support
- CLI tool (`iudex`) for running tests and generating reports
- Built-in governance and security checks for API endpoints
- Support for multiple HTTP frameworks:
  - Express.js integration
  - Native Node.js HTTP server support
- Database persistence with PostgreSQL
  - Test run tracking and history
  - Spreadsheet-style table view for results
- Multiple assertion types:
  - Status code validation
  - Response body schema validation (JSON Schema, Joi)
  - Header validation
  - Response time checks
- Test organization with tags and categories
- Detailed test reports with multiple output formats
- Dashboard server for viewing test results
- Variable interpolation in test definitions
- Support for test dependencies and execution order
